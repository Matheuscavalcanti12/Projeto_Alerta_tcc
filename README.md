🚗 Sobre o Projeto

O Projeto Alerta foi desenvolvido como Trabalho de Conclusão de Curso (TCC) com o objetivo de ajudar a prevenir acidentes causados por sono ao volante.

O sistema utiliza a câmera para monitorar o rosto do motorista em tempo real.
Através de técnicas de visão computacional, o programa identifica quando os olhos permanecem fechados por muito tempo.

Quando isso acontece, o sistema ativa um alerta sonoro, chamando a atenção do motorista e ajudando a evitar possíveis acidentes.

Este projeto demonstra a aplicação de tecnologia e programação na segurança no trânsito, utilizando processamento de imagem para identificar sinais de fadiga.
--
🛠️ Tecnologias Utilizadas

🐍 Python — linguagem principal do sistema

👁 OpenCV (cv2) — detecção facial e análise dos olhos

🗄 SQL — armazenamento de dados

🌐 HTML — estrutura da interface

🎨 CSS — estilização da aplicação

⚡ JavaScript — interações da interface

⚙️ Como Executar

Clone o repositório:

git clone https://github.com/seu-usuario/projeto-alerta.git

Instale as dependências:

pip install opencv-python numpy

Execute o sistema:

python detector_sono.py
